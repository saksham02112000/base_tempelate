<h2>For frontend</h2>
Open terminal (/base_template)
```
cd frontend
npm install
npm start
```
Make .env file with following configuration:
```
JWT_SECRET
```
<h2>For backend</h2>
Open terminal (/base_template)
```
cd backend
npm install
nodemon index
```
Make .env file with following configuration
```
REACT_APP_BASE_URL="URL at which Api should be called"
```
